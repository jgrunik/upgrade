## Developer Stuff

To install dependencies:

```bash
bun install
```

To serve:

```bash
bun run serve
```

To build:

```bash
bun run build
```

This project was created using `bun init` in bun v1.0.12. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
